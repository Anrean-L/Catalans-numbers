Числа Каталана

уникальная числовая последовательность, которая играет важную роль в различных областях математики. Эти числа имеют множество приложений, от организации данных в информатике до решения сложных математических задач.

Формула

Числа Каталана вычисляются по следующей формуле:

C3 = 5

Скобки

Количество правильных скобочных последовательностей, содержащих n пар скобок, равно числу Каталана Сn.

Например, если пар скобок 3, C3 = 5, значит и правильных скобочных последовательностей будет 5. И действительно:

()(()) ()()() (())()
(()()) ((()))

Рекуррентная формула

Рассмотрим правильные скобочные последовательности из (n + 1) пары скобок. Пусть закрывающаяся скобка, которая соответствует первой скобке, стоит на 2i месте, тогда между ними правильная скобочная последовательность из (i - 1) пары скобок, а после — последовательность из (n - i) пар

Значит, количество таких последовательностей из (n + 1) пары скобок равно сумме произведений количества последовательностей из (i - 1) пар и количества последовательностей из (n - 1) пар скобок. Получаем рекуррентную формулу чисел Каталана:

Примеры задач

Билеты стоят 50 центов, и 2n покупателей стоят в очереди в кассу. Половина из них имеет по одному доллару, остальные — по 50 центов. Кассир начинает продажу билетов, не имея денег. Сколько существует различных порядков в очереди, таких, что кассир всегда может дать сдачу?

Чтобы кассир мог дать сдачи, в любой момент времени среди покупателей, которые получили билет, должно быть не меньше человек с 50 центами чем с долларом. Заменим людей с 50 центами открывающими скобками, а остальных — закрывающими. Тогда такая очередь станет правильной скобочной последовательностью, а значит количество порядков равно Cn.
